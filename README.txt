Консольное приложение,
где в качестве интерфейса взаимодействия с пользователем мы используем консольное меню,
в котором у нас выводятся все возможные варианты действий доступных для пользователя.

Для функционирования консольного меню используется список фигур (хранится в ArrayList).

В качестве основных фигур выбраны:
* круг;
* прямоугольник;
* треугольник;

Вся общая логика по взаимодействию фигур вынесена в родиетльский класс Figure.

Что-бы в будущем было возможно добавление новых n-угольников, было предустмотрено создание класса, в котором вынесена общая логика n-угольников.

Для создания фигур в единообразном формате используется фабрика, в которой есть класс создателя с методом создания фигур и возвращением фигуры.
А так же три конкретных реализации создателей (круг, прямоугольник, треугольник).
